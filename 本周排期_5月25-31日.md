## 二、周六 · 赤水瀑布群拼团

### 活动定位

```
拼队测试案例。

```

### 拼队合作方案

| 项目 | 云途视界（我方） | 合作俱乐部（对方） |
|------|:---:|:---:|
| 路线 | ● 出路线（r041 赤水瀑布群） | |
| 领队 | ● 墨涵（摄影领队） | ○ 对方出一位领队（拼队标配） |
| 招募 | ○ 我方社群招募 | ● 对方主招募（他们有稳定报名量） |
| 摄影 | ● 墨涵全程拍 | |
| 成本 | 领队费用AA分摊 | 领队费用AA分摊 |
| 社群归属 | 对方的队员→引导关注云途 | 我方的队员→不主动导给对方 |

**合作原则：**
1. 拼队人数对半开，或按实际报名量灵活调整
2. 两家领队都参与带队，互相尊重路线决策权
3. 活动后在对方社群里不硬推云途，自然互动即可——人是慢慢过来的
4. 如果这次拼队体验好，后续形成固定合作机制
