# 中文注释技能

当用户调用此技能时，为当前的 AI 对话或代码添加中文注释。

## 指令

将以下内容逐段添加中文注释，保持原文 + 注释的格式：

### 注释规则

1. **技术术语** — 首次出现时用中文解释含义
   - 例: `通过 API (应用程序接口) 调用服务`

2. **代码片段** — 在关键行上方或行尾添加中文说明
   - 例: `const res = await fetch(url); // 发送异步请求获取数据`

3. **逻辑步骤** — 对每个处理步骤编号并用中文简述
   - 例: `步骤一：验证输入参数 → 步骤二：查询数据库 → 步骤三：返回结果`

4. **概念/缩写** — 展开缩写并解释
   - 例: `RAG (Retrieval-Augmented Generation，检索增强生成)`

5. **隐含假设** — 用方括号标注
   - 例: `[此处假设用户已登录]`

### 输出格式

```
【原文】
... 原始内容 ...

---

【中文注释版】
... 带有中文注释的内容 ...
```

### 注意事项

- 不修改原文内容，注释以附加形式呈现
- 注释简洁，每条不超过一行
- 对非技术人员友好，避免使用晦涩的专业名词二次解释
