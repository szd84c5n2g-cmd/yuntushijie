### 核心看点

| 看点 | 说明 |
|------|------|
| **拦马墙** | 古道险要处石砌防护墙，高约1米，类似现代公路护栏——千年"护栏"实物 |
| **千年古柏群** | 240余株，平均树龄1050年，最高2300年（状元柏），平均每25米一棵 |
| **青石板古道** | 石灰+糯米+桐油粘合，车辙痕、马蹄印、防滑线清晰可辨，石缝中可见古代工匠指纹 |
| **饮马槽/拴马桩** | 古代"加油站"和"停车场"实物 |
| **清凉桥** | 古道尽头标志性节点，适合合影 |
| **拦马墙乡村美术馆** | 百年老屋改建，可歇脚、写生 |
| **翠云廊** | Day2延伸段，"皇柏大道"，2025年获联合国"地球家园"范例奖 |

### 历史厚度（讲给队员的故事）

| 时期 | 事件 |
|------|------|
| 商周 | 金牛道已初步形成（2025年考古将开辟年代从战国推前至商代） |
| 战国 | 秦惠文王"石牛粪金"计诱蜀王开道，秦灭蜀 |
| 三国 | 刘备入川、诸葛亮北伐、姜维出征均经此道 |
| 秦代 | 始皇帝下令沿道植柏，称"皇柏" |
| 三国 | 张飞任巴西太守时命士兵植树，称"张飞柏" |
| 明代 | 剑州知州李璧植柏10万株，立"交树交印"制度 |
| 唐代 | 李商隐途经汉阳铺写下"君问归期未有期，巴山夜雨涨秋池" |

### 安全事项

| 类别 | 要点 | 细节 |
|------|------|------|
| 路况 | 青石板路 | 路面平整但部分石板有青苔，雨后湿滑，注意脚下 |
| 路况 | 古道窄段 | 部分路段宽仅2米，外侧有拦马墙保护，仍勿靠近边缘打闹 |
| 路况 | 上下坡 | 缓坡为主无陡崖，但连续石板下坡注意膝盖，登山杖有备无患 |
| 天气 | 防晒 | 古柏遮荫约60%，仍有阳光直射段，戴帽+防晒霜 |
| 天气 | 阵雨 | 5月底偶有阵雨，带轻型雨衣或折叠伞 |
| 天气 | 早晚温差 | 早晚15℃偏凉，带一件薄外套或皮肤衣 |
| 队伍纪律 | 跟队 | 古道岔路多，保持领队视线范围内 |
| 队伍纪律 | 摄影 | 古柏林中自由拍摄时间充足，但不要为找机位脱离主路 |
| 队伍纪律 | 信号 | 部分林间谷地信号弱，提前告知家人可能短暂失联 |
| 环保 | 垃圾 | **古道沿途垃圾桶稀少，所有垃圾随身带走** |
| 环保 | 古树保护 | 不攀爬古柏、不刻字、不折枝。千年古树需要每个人的尊重 |
| 环保 | 文物 | 不移动古道石块、不破坏青石板路面 |
| 人文 | 当地 | 途经村落请尊重当地生活，不喧哗、不擅入民居 |
