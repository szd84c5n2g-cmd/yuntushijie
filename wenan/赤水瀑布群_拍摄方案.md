## 一、瀑布拍摄"三有"原则

| 原则 | 为什么 | 怎么做 |
|------|--------|--------|
| **有人** | 空镜再美也只是风景照，有人才有代入感 | 每换一个瀑布，先拍景再拍人，人景比例 4:6 |
| **有尺度** | 瀑布的高大靠对比才能体现 | 人物站瀑布下方，用广角仰拍，人小景大 |
| **有情绪** | 报名冲动来自"我也想去这种状态" | 抓拍玩水瞬间、回头笑、水雾中闭眼的松弛感 |

## 二、照片：五个必拍镜头

### 1. 瀑布全景 + 人物渺小（广角 16-24mm）
- 构图：人物站瀑布正下方或侧面1/3处，镜头低角度仰拍
- 要点：人物穿亮色冲锋衣（橙/黄/红），在水雾暗背景中跳出来
- 用途：朋友圈首图、公众号封面

### 2. 慢门水流丝绸感（三脚架必备，ND滤镜）
- 快门 1/4s-2s，水流拉丝成白练
- 人物静止站立水中或岩石上，形成动静对比
- 用途：小红书首图，高级感拉满

### 3. 水雾中的逆光人像（50-85mm 大光圈）
- 利用瀑布产生的水雾做天然柔光
- 人物侧脸/背影，阳光穿透水雾形成光束
- 用途：朋友圈二图、人像吸引点

### 4. 第一人称涉水/溯溪（GoPro/手机防抖）
- 低角度跟拍队友穿行溪谷
- 脚踩水花、手拨竹叶的细节
- 用途：视频号/小红书视频，"想一起去"感

### 5. 团队氛围瞬间（手机即可）
- 休息时围坐吃干粮、帮队友拉一把过溪、水边大笑
- 不摆拍，抓真实互动
- 用途：社群预热、朋友圈配图，传递"这群人很好玩"

## 三、视频：三条硬核成片

### 视频 1：30s 沉浸式踩点 — "这条线太顶了"

| 项目 | 内容 |
|------|------|
| 目的 | 建立向往感，让人刷到就想转发 |
| 平台 | 小红书 + 视频号 |
| 格式 | 竖屏 9:16，30s |
| BGM | 纯环境音为主（水声+脚步+呼吸），高潮处淡入轻音乐 |

**分镜脚本：**

| 时间 | 画面 |
|------|------|
| 00-03s | 黑屏渐入水声 → 第一个瀑布全景（无人机俯拍/广角） |
| 03-08s | 人物从镜头前走过，走向瀑布（背影，人小景大） |
| 08-15s | 快速切换3个不同瀑布（每个2s），节奏加快 |
| 15-20s | 慢动作水花特写 + 人物踏水而过（关键帧变速） |
| 20-25s | 无人机拉升，峡谷全景 + 瀑布群同框 |
| 25-30s | 人物回头一笑定格 → 叠加文字"赤水瀑布群，下周出发" |

---

### 视频 2：15s 爽感切片 — "看完就想报名"

| 项目 | 内容 |
|------|------|
| 目的 | 小红书/抖音信息流直接转化 |
| 平台 | 小红书 + 抖音 |
| 格式 | 竖屏 9:16，15s |
| BGM | 节奏感强的卡点音乐 |

**四个卡点剪辑（每个 3-4s）：**

| 卡点 | 画面 |
|------|------|
| 🔥 1 | 人物纵身跳入水潭的瞬间（慢动作） |
| 🔥 2 | 瀑布从头顶浇下的第一人称视角（GoPro胸前支架） |
| 🔥 3 | 队友在水雾中大笑的慢镜头 |
| 🔥 4 | 无人机快速穿越瀑布水雾 → 定格 + "报名入口在主页" |

---

### 视频 3：60s 氛围长片 — "这个社群值得加入"

| 项目 | 内容 |
|------|------|
| 目的 | 展示团队氛围，让人想"和这群人一起玩" |
| 平台 | 视频号 + B站 |
| 格式 | 横屏 16:9，60s |

**分镜脚本：**

| 时间 | 画面 | 旁白/字幕 |
|------|------|----------|
| 0-15s | 集合出发、车上聊天、分发装备 | "周五晚上还坐在工位，周六早上就站在瀑布面前" |
| 15-35s | 徒步穿行竹林、过溪、攀爬岩石 | "没有KPI，只有下一个水潭有多清" |
| 35-50s | 水潭边休息、吃干粮、互相拍照 | "领队会帮你找最好的机位" |
| 50-60s | 夕阳返程，大巴上大家睡着 | "这就是我们的周末。你要来吗？" |

## 四、视频拍摄「三个一」技法

### 一个稳定器级别的替代方案
- 没有稳定器？用相机/手机背带绷紧靠脖子，双手持机，走"忍者步"（膝盖微曲，脚掌滚动着地）
- 瀑布边石头滑，优先保命再保镜头

### 一个必须拍的镜头：穿越水幕
- GoPro 从瀑布水帘外侧走进内侧（或反过来）
- 这是瀑布视频里互动感最强的镜头，一条就值回票价
- 注意保护好设备，拍完立刻擦干

### 一个最容易忽视的镜头：声音
- 单独录 30s 纯瀑布水声（环境音），后期铺在慢镜头下
- 比任何 BGM 都有质感
- 再录一段团队笑声/说话声，用于氛围片

## 五、设备分工

| 角色 | 设备 | 主拍内容 |
|------|------|---------|
| 墨涵（摄影向） | 相机+广角+长焦+三脚架 | 慢门水流、人像、全景素材 |
| 喜帖街（氛围向） | GoPro + 手机 | 第一人称涉水、卡点视频、花絮抓拍 |
| 共享 | 无人机（如有） | 峡谷俯拍、瀑布群全景、团队行进 |

> 拍摄时墨涵先站位构图，喜帖街带人走位，效率最高。

## 七、设备 Checklist

```
□ 三脚架（慢门必备）
□ ND滤镜（白天长曝光）
□ 广角镜头（16-35mm）
□ 中长焦（70-200mm 或 85mm定焦）
□ GoPro/运动相机（涉水镜头）
□ 手机防水袋
□ 镜头擦拭布（水雾会一直打湿镜头！）
□ 备用电池（水雾环境耗电快）
□ 无人机（赤水峡谷俯拍瀑布群全景绝佳）
□ 亮色服装（出镜领队必穿亮色）
```
