来回5公里，边走边玩的那种，不是军训。

## 第三波：一对一私信（5/27 周三晚）

挑5-8个之前参加过活动、比较活跃的老队员，私信：

```
[对方名字]，周四我组了个自己人玩的小局。

去沙河坝轻徒步+河边做竹筒饭，还有个舞蹈老师朋友来。
不赚钱的局，AA，就8-15个人。

想叫你一起，感觉这种氛围你会喜欢。
来的话跟我说一声 😄
```

> 要点：私信不要群发感，带上对方名字，表达"这个局适合你"而不是"我在拉人头"。

## 文案风格说明

| 原则 | 说明 |
|------|------|
| 口语化 | 用"组个局"而不是"举办活动"，用"自己人"而不是"社群成员" |
| 透明 | 直接说"不赚钱""AA""人均三四十块"，不用绕 |
| 低门槛 | 反复强调"不用有基础""会笑就行"，消除参与顾虑 |
| 不做作 | 不写emoji轰炸、不喊口号、不用感叹号结尾 |
| 私密感 | 不发小红书不发朋友圈公开招募，只在群里说，一对一私信也强调"自己人的局" |
